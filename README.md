# HELMET-DETECTION
 To monitor whether the two-wheeler riders wears helmet while driving and to fine a particular amount once they are seen violating the rule by riding without helmet.
 
# To run the project:
 1.Install the following packages:
 ```
    . Tensorflow
    . Keras
    . Opencv
    . imutils
    . numpy
    . time
    . h5py
  ```
